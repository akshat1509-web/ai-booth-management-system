# AI-Driven Booth Management System

## Overview

This project transforms static voter lists into an **AI-powered knowledge graph platform** that enables hyper-local governance insights and targeted communication.

The system helps administrators understand voter demographics, detect local issues, and deliver personalized governance updates at the booth level.

---

## Problem

Traditional voter data is stored as static spreadsheets. This creates several challenges:

- No intelligence or insights from voter data
- Difficulty identifying issues at the booth level
- Lack of targeted citizen communication
- Limited integration with government schemes

---

## Solution

Our system converts voter datasets into a **dynamic knowledge graph** that connects:

Voters → Booths → Issues → Government Schemes

Using AI segmentation and graph analytics, the platform enables:

- booth level analytics
- voter segmentation
- issue detection
- hyper-local communication

---

## Key Features

• AI-based voter segmentation  
• Knowledge graph visualization  
• Booth-level analytics dashboard  
• Hyper-local governance targeting  
• Civic intelligence insights
